# scratchon
A Python and Scratch Project Connector!

### Installation
```
pip install scratchon
```

### Get started
How to set up a client connection:

```Python
import scratchon

# Instantiate a client object
account = scratchon.Client('scratch_username', 'scratch_password')
```
